# DarkWatch

DarkWatch is a cybersecurity project focused on monitoring dark web resources for organization-specific keywords. The platform is designed with a secure microservice architecture that separates the user-facing application from the scanning service to reduce security risks and follow secure software engineering principles.

## Project Overview

Organizations need early visibility into potential data exposure on underground platforms. DarkWatch allows users to register keywords such as company names, product names, employee names, or internal project names. The system scans configured dark web resources through an isolated service and reports potential matches to the main application.

This project is being developed as part of a cybersecurity software engineering internship and emphasizes secure architecture, defensive monitoring, and modern software engineering practices.

## Planned Technology Stack

- **Frontend:** React
- **Backend:** Node.js + Express
- **Scanner Service:** Python
- **Database:** PostgreSQL
- **Queue:** Redis
- **Containerization:** Docker
- **Network:** Tor
